
# :memo: CRUD

Folder ini berisi penerapan update dan delete data dengan bahasa pemrograman PHP. Materi yang dibahas meliputi:
* Mengubah data
* Menghapus data

## :package: Prasyarat

Sebelum memulai, pastikan telah terinstall:
* MySQL atau MariaDB
* PHP 5 atau PHP 7
* Text editor
* Web browser
* Web server
* Materi P3-CRUD
