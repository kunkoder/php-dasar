
# :outbox_tray: Upload

Folder ini berisi penerapan upload file dengan bahasa pemrograman PHP. Materi yang dibahas meliputi:
* Mengupload file
* Menghapus file

## :package: Prasyarat

Sebelum memulai, pastikan telah terinstall:
* MySQL atau MariaDB
* PHP 5 atau PHP 7
* Text editor
* Web browser
* Web server
* Materi P4-CRUD
